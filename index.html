How Portify Works

A simple and reliable way to build your portfolio.

1. Choose a Template

Browse templates and preview the design before downloading.

2. Download the HTML File

Save the template to your system. The file is fully editable.

3. Edit Using Your CV

Open the file in VS Code or any text editor and replace the content with your personal details, education, skills, projects, and contact information.

Deploy on GitHub Pages

Free hosting with a clean, professional link.

Create a Repository

Create a new repository on GitHub.

Upload Your File

Upload the edited HTML file to the repository.

Enable GitHub Pages

Go to Settings → Pages, select the main branch, and save.
